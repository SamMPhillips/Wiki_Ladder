Vertex * WikiTree::buildWikiTree ( const std::string & stopUrl)

Builds a WikiTree from the root member variable until a Vertex with a specified key value is found. Returns the Vertex.

Usage: Vertex *end = buildWikiTree(const std::string &stopUrl);

Function builds a WikiTree, a graph of Vertex objects with Wikipedia page urls as their key values. While building searches for a Vertex with stopUrl as its key value.

Parameters
stopUrla reference to a const std::string.
Returns
Vertex *end, a pointer to the Vertex with key variable of stopUrl. If Vertex is not found, returns a pointer to the root Vertex.
